What is Browser?

A browser (short for web browser) is a software application used to access and view websites and content on the World Wide Web. It retrieves information from web servers and displays it to users in a readable and interactive format

Google Chrome , Mozilla Firefox , Safari (by Apple) , Microsoft Edge , Opera

What is URL

URL (Uniform Resource Locator) is the address used to identify and locate a specific resource on the internet. A URL is a reference (an address) to a resource on the Internet. It tells the browser where to find a resource and how to retrieve it.

Example: https://www.wikipedia.org

What is Domain?

A domain is a user-friendly name used to identify a website or a group of computers within a network. It simplifies access by mapping to IP addresses or centralized resources

Example: Website domain – google.com; Network domain – corp.company.com.

What is IP Address?

An IP (Internet Protocol) address is a unique numerical label assigned to each device connected to a network that uses the Internet Protocol for communication.

Example: 192.168.1.1 (commonly used for home routers).

Difference between markup language and programming language?

A markup language is used to structure and present content on the web (e.g., HTML). A programming language is used to write instructions that a computer can execute (e.g., Python, Java).

Example: Markup Language:Hello (HTML) Programming Language: print("Hello") (Python)

What is a Bug?

A bug is an error or flaw in software code that causes a program to behave unexpectedly or incorrectly.

Example: A login page that doesn't accept the correct password due to faulty code has a bug.

What is Debug?

Debug is the process of identifying, analyzing, and fixing bugs or errors in software code.

Example: A developer uses debugging tools to find why a button isn’t working and then fixes the issue in the code.
